mcprotocol
==========

Botting API for Minecraft using the client-server protocol





Format Specifiers
\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_
|b| one byte    |
|h| two bytes   |
|w| four bytes  |
|l| eight bytes |
|q| 16 bytes    |
|s| string      |
|v| varint32    |
|*| array       |
|_|_____________|
